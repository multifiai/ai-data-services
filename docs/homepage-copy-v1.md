# Homepage Copy V1

## Copy Direction

**Working brand motif:** Data Foundry

This first copy pass is written for the selected visual direction: crisp AI infrastructure UI over a dark, impressionist data atmosphere. The tone should feel premium, precise, and trustworthy, with enough imagination to match the Monet Dark visual system.

The copy avoids copying Abaka directly. It positions the company as a human-led data infrastructure partner for frontier AI teams: dataset creation, expert annotation, evaluation, RL environments, and secure delivery.

## 3. Data Lifecycle

**H2:** From Raw Signals to Reliable Training Data

**Supporting copy:** A controlled production workflow turns messy real-world inputs into verified datasets that models can trust.

**Lifecycle steps**

**1. Collect**  
Capture multimodal source signals from real use cases, domains, and workflows.

**2. Curate**  
Filter, normalize, deduplicate, and prepare data for expert work.

**3. Annotate**  
Apply structured human judgment with guidelines, tooling, and review logic.

**4. Review**  
Resolve ambiguity with QA layers, consensus checks, and specialist escalation.

**5. Evaluate**  
Measure quality, model readiness, and coverage across critical dimensions.

**6. Deliver**  
Package verified datasets for training, evaluation, or agent reinforcement loops.
